# Bipolar Awareness Website

A clean, informative single-page website about bipolar disorder — built for awareness, understanding, and support.

## Features

- Animated mood wave hero
- Key statistics and episode type breakdowns
- Bipolar I, II, Cyclothymia, and Other Specified explanations
- Interactive mood check-in
- Treatment & self-management resources
- Crisis helpline information (988, Crisis Text Line)

## Usage

Open `index.html` in any browser:

```bash
xdg-open index.html
# or
firefox index.html
```

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set branch to `main` → Save
4. Live at `https://YOUR_USERNAME.github.io/bipolar-awareness`
